 # PHYS389-CODE

This code served as the repository for the module PHY389--Computer programming and modelling. 

The code defiend a class that returns the decay rate of any isotppe given its properties (initial conditions, decay rate, decay constant, etc). 

The class is composed of three diffrent functions:
- __init__: initialization function
- final_quantity: computes the dinal isotope quantity after decay given a certain time
- decay_rate: calculates the decay rate

The class returns the functions above and can then be plotted as desired.
